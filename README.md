<h1 align="center">📚 Library Management System</h1>

<p align="center">
  A web-based Library Management System for local libraries, allowing users to check book availability and submit reservation requests online before picking up books in person.
</p>

---

## 🌟 Features  

✅ **Search & Browse Books** – Find books easily with an interactive search.  
✅ **Availability Check** – See if a book is currently available.  
✅ **Reservation Requests** – Request books online and collect them at the library.  
✅ **User Accounts** – Library members can sign up and track their reservations.  
✅ **Admin Dashboard** – Manage books, users, and reservations efficiently.

## 🏗 How to Set Up Locally  

```bash
# Clone the repository
git clone https://github.com/yourusername/library-management-system.git

# Navigate into the project
cd library-management-system

# Install backend dependencies (Laravel)
composer install

# Install frontend dependencies (React)
npm install

# Set up the database (MySQL)
php artisan migrate

# Start the backend server
php artisan serve

# Start the frontend
npm start
